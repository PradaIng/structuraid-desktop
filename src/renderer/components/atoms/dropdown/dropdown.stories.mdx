# Dropdown

##  Usage

This is `Dropdown` atom, used to insert an selectable list of items in the UI.

The component supports being used in a light or dark background, specifying the `color` prop:

- `light`
- `dark`

This is the default style, it is compact, use it when the component is being used along with other inputs in a long form
