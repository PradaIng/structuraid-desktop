# Button

This is `Button` atom. Structuraid has tree option, defined with `as` prop:

- `primary`
- `secondary`
- `warning`

And `variant` prop:

- `big`
- `small`

Also, can be used one of thes options for `useTag`prop:

- `button`
- `anchor`
