# Button


##  Usage for App UI

This is `Button` atom, used to insert text in our software UI. 
We have 2 types of interface's text, indicated by `type` prop:

- `primary`
- `secondary`

And `variant` prop:

- `big`
- `big-full-w`
- `small`
- `small-full-w`
